# SparkToDo

A minimalist screen-highlight tool. Hold a modifier key + left-click drag to draw a rainbow-striped rectangle on any screen area -- release to dismiss. Perfect for emphasizing content during screen recordings.

## Installation

### One-liner (recommended)

```powershell
irm https://raw.githubusercontent.com/WodenJay/HoldRect/main/install.ps1 | iex
```

### Manual download

1. Go to [Releases](https://github.com/WodenJay/HoldRect/releases/latest)
2. Download `holdrect.exe`
3. Run it
